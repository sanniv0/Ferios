# Ferios - Beautiful Animated React Components

Ferios is an open-source collection of high-quality, animated, interactive & fully customizable React components for building stunning, memorable user interfaces. This component library provides developers with a suite of beautifully crafted UI elements that come with smooth animations and elegant designs out of the box.

## Component Library

The Ferios component library includes:

### Buttons
- Animated Button: Gradient button with shimmer animation effect

### Cards
- Glow Card: Card component with animated gradient border on hover

### Inputs
- Toggle Switch: Smooth animated toggle switch with gradient track
- Input Field: Modern input field with animated floating label and validation states
- Dropdown: Customizable dropdown menu with smooth animations

### Display
- Avatar Stack: Overlapping avatar group with hover animation
- Notification Badge: Icon button with animated notification count badge
- Tabs: Animated tab navigation with smooth indicator transitions

### Feedback
- Progress Bar: Animated progress bar with gradient fill and shimmer
- Skeleton Loader: Animated placeholder for loading content states

### Overlay
- Tooltip: Elegant tooltip with smooth fade and position animations
- Modal: Elegant modal dialog with smooth entrance animation and overlay backdrop

## Development Scripts

- `npm run dev` - Starts the development server
- `npm run build` - Builds the application for production
- `npm run preview` - Locally preview the production build
- `npm run lint` - Runs ESLint to check for code issues

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Start the development server with `npm run dev`
4. Visit `http://localhost:5173` in your browser

## Customization

Each component is designed to be highly customizable:
- Modify CSS variables in `src/index.css` for global theming
- Override component props for individual customization
- Use the component code as a starting point for your own variations
